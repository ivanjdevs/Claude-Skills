def fig_to_b64(fig):
    buf = BytesIO()

### Fase 1 — Carga y Overview

```python
# Detectar formato por extensión
if FILE_PATH.endswith(('.xlsx', '.xls')):
    df = pd.read_excel(FILE_PATH)
else:
    try:
        df = pd.read_csv(FILE_PATH, encoding='utf-8')
    except UnicodeDecodeError:
        df = pd.read_csv(FILE_PATH, encoding='latin-1')

n_rows, n_cols = df.shape
num_cols  = df.select_dtypes(include='number').columns.tolist()
cat_cols  = df.select_dtypes(include='object').columns.tolist()

# Detectar columnas datetime (por tipo o por nombre)
dt_cols = df.select_dtypes(include='datetime64').columns.tolist()
for col in df.columns:
    if col not in dt_cols and any(k in col.lower() for k in ['date','fecha','time','timestamp']):
        try:
            df[col] = pd.to_datetime(df[col])
            dt_cols.append(col)
        except:
            pass
```

Narrativa de esta fase: forma del dataset, tipos de columnas, si hay fechas detectadas.

### Fase 2 — Calidad de Datos

Calcular y presentar en una **tabla resumen** (sin heatmap ni gráficas):
- **Nulos:** columna con cantidad y % de nulos por variable, ordenada de mayor a menor %. No incluir columna de severidad/badges — el número y porcentaje ya son suficientes.
- **Duplicados:** `df.duplicated().sum()`
- **Columnas sin varianza:** columnas con un único valor único.
- **Tipos inconsistentes:** columnas `object` que podrían ser numéricas.

Narrativa: mencionar problemas encontrados y su impacto potencial (clasificar verbalmente en la narrativa si una columna es crítica, ej. "> 30% nulos", pero sin columna de badge en la tabla).

### Fase 4 — Análisis Univariado

### Fase 5 — Detección de Outliers

Para cada columna numérica:
```python
Q1, Q3 = df[col].quantile([0.25, 0.75])
IQR = Q3 - Q1
outliers_iqr = ((df[col] < Q1 - 1.5*IQR) | (df[col] > Q3 + 1.5*IQR)).sum()

z_scores = np.abs(stats.zscore(df[col].dropna()))
outliers_z = (z_scores > 3).sum()
```

Generar **solo** la tabla resumen con conteo (y %) por método, una fila por columna numérica. No generar boxplots ni gráficas para esta fase. Narrativa: señalar columnas con outliers significativos (> 5% de filas) y sugerir revisión manual.

### Regla obligatoria para los hallazgos clave

Cada hallazgo debe incluir **al menos un número concreto** que lo respalde. No se permiten afirmaciones cualitativas sueltas (ej. "correlación fuerte", "distribución sesgada", "varios outliers") sin el dato que las sostenga.

Checklist según el tipo de hallazgo:

| Origen del hallazgo | Número que debe incluir |
|---|---|
| Correlación (Fase 6/7) | Valor de r (ej. "r ≈ 0.79") |
| Skewness (Fase 3/4) | Valor numérico exacto (ej. "skewness ≈ 1.88") |
| Nulos (Fase 2) | % exacto por columna, no solo el conteo de columnas afectadas (ej. "PoolQC: 99.5% nulos") |
| Outliers (Fase 5) | % de filas afectadas y/o columnas específicas (ej. "GrLivArea: 6.2% de outliers") |
| Comparación categórica (Fase 6/7) | Valores concretos de cada grupo comparado (ej. "NridgHt: mediana $300K vs MeadowV: $105K") |
| Multicolinealidad (Fase 8) | Valor de r entre el par de variables (ej. "GrLivArea vs 1stFlrSF: r = 0.93") |
| Feature engineering (Fase 8) | El número que motiva la recomendación (ej. "5 valores únicos → candidata a binning") |

Mal ejemplo: *"Las variables más correlacionadas con el precio son las de calidad y tamaño."*
Buen ejemplo: *"`OverallQual` tiene la correlación más alta con `SalePrice` (r ≈ 0.79); una vivienda de calidad 10 cuesta en promedio más de 3× que una de calidad 3."*

## Reglas generales

- **Narrativa en el idioma del usuario.** Si el usuario escribió en español, todo el HTML va en español.
- **Columnas con > 20 categorías únicas:** omitir de visualizaciones pero mencionar en el reporte.
- **Datasets grandes (> 100k filas):** usar `df.sample(5000, random_state=42)` para pairplots y scatter plots, pero calcular estadísticas sobre el total. Indicarlo en el reporte.
- **Figuras:** siempre `plt.close(fig)` después de convertir a base64 para liberar memoria.
- **Títulos de gráficos:** siempre incluir título descriptivo y labels en ejes.
- **`matplotlib.use('Agg')`:** primera línea después de `import matplotlib`, sin excepción.
